# README terminal demo

GitHub READMEs cannot play video tags reliably, so the animated terminal in the
[root README](../../README.md) is a **GIF** (generated by
[VHS](https://github.com/charmbracelet/vhs) from a `.tape` script). Alternatives
like asciinema `.cast` files need an external player.

```bash
# from repo root
./docs/demo/record.sh
```

Requires `vhs`, `ffmpeg`, `ttyd`, and `nix` on `PATH`.

## What the GIF shows

1. `nxr list` / inline `flake#app` run (`fixtures/basic-apps`)
2. `inspect`, `graph --format mermaid`, `graph --format dot` (`fixtures/task-dag`)
3. Task **alias** `check` → `ci` with `--dry-run` (schedule + exact `nix run` lines)
4. Parallel `task join -j 2` with `--output live` (`fixtures/parallel-group`)
5. `--shell default` (`fixtures/named-dev-shells`)
6. `watch` start and Ctrl-C shutdown

Recording suppresses Nix dirty-tree warnings and warms fixture eval caches so the GIF stays readable.
| File | Role |
|---|---|
| [nxr.tape](nxr.tape) | Recording script |
| [nxr.gif](nxr.gif) | Output embedded in the root README |
| [record.sh](record.sh) | Build `nxr` + run VHS |
